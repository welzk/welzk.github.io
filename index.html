Die Kunst der Schwärme

Description

Die Kunst der Schwärme is a research project that explores the mathematical simulation of flocks, accompanied by an interactive installation.

Skills

  • Touchdesigner

Credits

Prof. Jan Bernkopf

Die Kunst der Schwärme (The Art of Swarms) is a research project and interactive installation focused on the mathematical simulation of flocking behavior. The project uses the Boids model, which is based on three simple rules—separation, alignment, and cohesion—that allow individual "boids" to create complex, collective movement.The interactive installation, built with TouchDesigner, uses a GLSL Shader to handle the thousands of calculations needed for the simulation. Users can manipulate the flock's behavior in real time, even simulating predator-prey scenarios, using hand and pose recognition via the MediaPipe plugin.

Invisible Intruder

Description

Invisible Intruder is an interactive exhibition prototype, visualizing data flow and cybersecurity risks of smart home devices, to educate users.

This project was part of my European Project Exchange Semester at the Oslo Metropolitan University.

Skills

  • international teamwork
  • laser cut
  • 3D print
  • Arduino

Credits

Dinis Costa, Joep van der Meulen, Ivan Peña Carreño, Luca Palaysi and Kevin Welz

“Invisible Intruder” is an interactive exhibition, developed to increase the awareness of
cybersecurity risks in smart homes. As the usage of smart devices keeps growing, many users
aren’t aware of how this technology handles personal data or how easily hackers can access
those and have a real-life impact. To address this challenge, we created a physical miniature
of a smart home in combination with a videogame, where the participant takes the role of a
hacker to breach into a home through the smart device network.
